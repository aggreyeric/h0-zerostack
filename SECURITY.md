# Security Policy

## Reporting a Vulnerability

Found a security issue? **Do not open a public GitHub issue.**
Report it privately via GitHub Security Advisories
(*Report a vulnerability* on the repo) or email the maintainers directly.

Please include: description, reproduction steps, and impact assessment.

## Supported Versions

We patch only the latest release. Older versions receive no backports.

## Response

ZeroDeploy is a hackathon project maintained by volunteers.
There is **no SLA**; we aim to acknowledge reports within a reasonable
timeframe. Responsible disclosure is appreciated.
